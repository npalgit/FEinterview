# 选择器
```css
1. * 通用选择器
2. . class
3. # id
5. div 标签选择器
---
6. X Y 后代
7. X>Y 子代
---
8. X+Y 直接兄弟：第一个兄弟节点
9. X~Y 兄弟：所有兄弟节点
---
10. [attr] 设置了attr属性的元素
11. [attr=value] 属性值刚好为value
13. [attr~=value] 属性值中，其中一个为value
14. [attr|=value] 属性值value， value开头
---
15. [attr^=value] 属性值value开头
16. [attr$=value] 属性值value结尾
17. [attr*=value] 属性值包含value
```
