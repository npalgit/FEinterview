## 编写一个方法 求一个字符串的字节长度

假设：一个英文字符占用一个字节，一个中文字符占用两个字节
```
 function GetBytes(str){

        var len = str.length;

        var bytes = len;

        for(var i=0; i<len; i++){

            if (str.charCodeAt(i) > 255) bytes++;

        }

        return bytes;

    }

alert(GetBytes("你好,as"));
```

## js对象的深度克隆
```
function clone(obj) {
    let buf; 
    if (obj instanceof Array) {
        buf = [];
        for (let i in Array) {
            buf[i] = clone(obj[i]);
        }
        return buf;
    } else if (obj instanceof Object) {
        buf = {};
        for (let i in obj) {
            buf[i] = clone(obj[i]);
        }
        return buf;
    } else {
        return obj;
    }
}
```
## 数组去重
```
Array.prototype.unique = function() {
    var n = [];
    for (let i of this) {
        if (n.indexOf(i) !== -1) {
            n.push(i);
        }
    }
}
```
