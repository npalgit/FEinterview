### 2. 建立作用域链
- 当前环境+上层环境变量对象
- 保证当前执行环境对符合访问权限的变量和函数有序访问
>闭包
- 闭包：函数可以记住并访问其所在的作用域
- 通过闭包，可以在其他执行上下文访问到函数内部变量
- 闭包用途：setTimeout, 柯里化， 模块

### 3. 确定this的指向
- this是在函数被调用时确定的
- this由调用者提供，由调用方式决定

>1.确定this
- 找到函数调用者（对象）
- 是否独立调用

>2.修改this指向
- apply(obj, 数组)
- call(obj, xx, xx, xx)

>3.应用
- arguments->array

Array.slice.call(arguments)
- this指向
- 继承
```js
function SubT(name, age) {
    SuperT.call(this, name);//借用构造，继承属性
    this.age = age;
}
```
